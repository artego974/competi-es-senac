# Gestão de orndens de Serviço Técnicas

## Front-End

As ferramntas ultilizadas para o projeto serão html css e javascript vanilla 

## Back-End 

As ferramentas ultilizadas para o projeto serão node express typeorm 

## Banco de Dados

A logica do DER foi feita na ferramenta canva e mostra as 5 tabelas principais 

1 cliente pode ter varias ordens e uma orden pode ter apenas um cliente

1 cliente pode ter varias equipamentos e um equipamento pode ter apenas um cliente

1 técnico pode pode ter varios ordem e 1 ordem pode ter 1 técnico

1 equipamento pode ter 1 ordem e 1 ordem pode ter varios equipamentos
